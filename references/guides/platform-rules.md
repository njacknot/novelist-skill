# 平台规则与商业投稿指南

> 商业网文投稿时参考本文件。自娱创作可忽略。
> 覆盖：主流平台的推荐机制要点 / 违禁词分级 / 章节命名 / 标签分类。

---

## 1. 主流平台速查

| 平台 | 核心机制 | 前三章要求 | 单章字数 | 日更期望 |
|------|---------|-----------|---------|---------|
| 番茄小说 | 推荐评估 + 完读率 | 首章必爆点，3 章定主线 | 2000-3000 | 6000-10000 |
| 起点中文 | 新书榜 + 编辑推荐 | 黄金三章，世界观克制 | 2500-3500 | 4000-6000 |
| 七猫 | 用户留存率 | 首章钩子 + 金手指曝光 | 2000-3000 | 6000-8000 |
| 晋江 | 积分 + 收藏增速 | 文笔优先，情绪密度高 | 3000-4000 | 2000-4000 |
| 微信读书 | 书架留存率 | 前 3 章决定留存 | 3000-5000 | 不强制 |

### 通用前三章硬指标

- 首章 **开头 300 字内**建立即时冲突或身份悬念
- 首章内让主角身份显现、金手指/金线曝光
- **第 3 章内**讲清故事主线与读者期待
- 前 3 章每章结尾必须有强钩子

详见：[reader-simulator-spec.md](../advanced/reader-simulator-spec.md) §开篇三章专项。

## 2. 违禁词分级

> 违禁词随政策变化。本表只列**长期稳定**的分级规则。具体平台最新词库需以平台公告为准。

### P0 · 绝对禁用（一票否决）

- 涉政/涉领导人/敏感历史事件
- 涉黄直接描写（性行为具体过程）
- 涉毒品制作 / 具体教唆犯罪方法
- 自残/自杀的操作性描写

### P1 · 高风险（需替代表达）

| 类别 | 示例原词 | 替代策略 |
|------|---------|---------|
| 暴力血腥 | 爆头、碎尸、鲜血喷涌 | 改为“倒下”“不再动弹” |
| 宗教 | 真主、上帝、佛祖直接引用 | 改为虚构神祇名 |
| 迷信 | 算命、风水的操作描写 | 改为“某种直觉” |
| 国家/民族 | 具体现实国家与民族冲突 | 改为架空国名 |

### P2 · 中风险（轻度改写）

- 烟酒描写：减少具体品牌，减少细节
- 赌博描写：不写具体赔率与赌局推进
- 外貌歧视词：统一中性化

### 落实到写作流程

- 初稿阶段不强制过滤，避免抑制表达
- 发布前统一执行一次替换扫描（可在 `/检查 --校稿` 中并入）
- 长期出现的敏感类型，写入项目级 `banned_words.local.md` 覆盖本表

## 3. 章节命名规范

### 文件名

- 格式：`第{NN}章-{章节标题}.md`
- 章号两位以上补零（第 01 章、第 100 章）
- 不使用：空格、标点、英文引号

### 章节标题本体

| 平台 | 偏好 |
|------|------|
| 番茄 | 短标题（4-8 字），结果型/悬念型，例：“她的刀出鞘了” |
| 起点 | 中等（6-12 字），留白型，例：“不速之客” |
| 晋江 | 情绪型（4-10 字），例：“他到底在想什么” |

### 禁止模式

- 剧透标题（“主角打败了XX”）
- 过长标题（> 15 字）
- 编号重复或乱序

## 4. 标签与分类

> 标签决定平台推荐池，错标等于错过目标读者。

### 建议方法

1. 选 **1 个主分类**（玄幻/都市/言情/科幻/悬疑/历史）
2. 选 **2-3 个子标签**（如“重生 + 都市异能 + 轻松”）
3. 标签必须与开篇三章兑现一致，不可开篇写正剧打重生标签

### 常见错配

| 表现 | 风险 |
|------|------|
| 标“轻松”实写“压抑” | 用户跳出率高 |
| 标“爽文”实写“慢热” | 差评集中 |
| 标“重生”但金手指不是重生 | 推荐失效 |

## 5. 简介与封面

### 简介公式（通用）

```
身份反差（1 句）
+ 核心事件（1 句）
+ 世界观钩子（1 句）
+ 卖点承诺（1 句）
```

简介总长控制在 **80-150 字**。

### 封面

- 主角占比 > 60%
- 颜色层次清晰，移动端缩略图可辨
- 不使用明显 AI 痕迹（多手指、脸部歪斜）
