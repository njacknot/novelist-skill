## 一、骚话八大分类

### 1. 装逼打脸型（A）

**定义**：主角实力碾压或逆转时，用轻描淡写的台词制造反差爽感。

**核心手法**：轻描淡写法 / 借力打力法 / 数据碾压法

| 编号 | 场景 | 模板 | 示例 |
|------|------|------|------|
| A1 | 实力碾压 | "你的{强项}，还不如我的{弱项}" | "你的全力一击，还不如我打个喷嚏" |
| A2 | 逆转打脸 | "（重复对方嘲讽）+ 所以呢？" | "你说没人能接你三招？我只用了一根手指" |
| A3 | 不屑一顾 | "{轻松动作} + 你们继续" | 打了个哈欠："你们继续，我先眯一会" |
| A4 | 事后补刀 | "差点忘了告诉你 + {真实实力}" | "差点忘了告诉你，我刚才是左手" |
| A5 | 境界碾压 | "你修炼{时长}的功法，我翻了一遍目录" | "你苦修五十年的剑意？哦，我翻了翻入门手册" |
| A6 | 数量碾压 | "你觉得{数量}很多？" | "一千万灵石？我以为你要说一个亿呢" |
| A7 | 无视攻击 | "{被攻击后的日常动作} + 你打完了？" | 拍了拍肩上的灰："你打完了？该我了" |
| A8 | 借物打脸 | "这个{对方珍视的东西}，在我那只能{低级用途}" | "你的灵宝？在我那用来垫桌脚" |
| A9 | 身份碾压 | "{低调自述} + {真实身份暴露}" | "我师父？他老人家不太出名……你们叫他剑圣" |
| A10 | 时间碾压 | "你用了{长时间}，我只需要{极短时间}" | "你参悟十年的阵法，我看了一炷香就破了" |
| A11 | 预言打脸 | "我说过{预言}，你不信。现在呢？" | "我说过三招之内你会跪下。现在第二招，你要不要提前？" |
| A12 | 让步打脸 | "我让你{优势}，再来" | "我闭着眼再打一次，你可以叫帮手" |

**节奏要求**：打脸前必须有 ≥1 段铺垫（对方嚣张 → 围观者绝望 → 反转），禁止无铺垫直接装逼。

### 2. 毒舌吐槽型（B）

**定义**：犀利讽刺、黑色幽默、冷面杀手式的语言输出。

**核心手法**：精准类比法 / 假装好心法 / 冷面叙述法

| 编号 | 场景 | 模板 | 示例 |
|------|------|------|------|
| B1 | 嘲讽对手 | "你{动作}的样子，像极了{反差物}" | "你挥剑的样子，像极了在赶苍蝇" |
| B2 | 回怼挑衅 | "建议你{看似正常} + {毒舌原因}" | "建议你别说话了，你一开口就拉低了整条街的智商" |
| B3 | 旁观评价 | "{冷静客观} + {突转毒舌}" | "客观来说，你的修炼天赋整个宗门倒数——哦不，倒数第一是门口的看门狗" |
| B4 | 假装关心 | "我真担心你{关心内容}——{毒舌反转}" | "我真担心你的伤势——万一没死透怎么办" |
| B5 | 冷面陈述 | "{用学术/官方语气描述丢人的事}" | "根据我有限的观察，阁下的逃跑速度是唯一达到宗师级的能力" |
| B6 | 帮倒忙式 | "别谢我，我只是{表面帮忙}" | "别谢我提醒你，你的裤子破了——大概是被你膨胀的自信撑破的" |
| B7 | 比较伤害 | "和{更差的东西}比，你还算{微弱肯定}" | "和石头比起来，你还算有点表情" |
| B8 | 精确量化 | "你的{能力}，精确相当于{荒谬量化}" | "你的战斗力，精确来说相当于一只营养不良的鸡" |
| B9 | 反问连击 | "{反问1}？{反问2}？还是{最毒反问}？" | "你是眼瞎？还是脑子进水了？还是两个同时发作？" |
| B10 | 翻译体 | "翻译一下刚才这位说的话：{毒舌翻译}" | "翻译一下：'你不配和我动手'的意思是'我打不过你但嘴硬'" |

### 3. 撩人情话型（C）

**定义**：暧昧、撩拨、让人心跳加速——不是土味情话，而是贴合人物性格和情境的高级撩。

**核心手法**：言外之意法 / 反撩法 / 场景借力法

| 编号 | 场景 | 模板 | 示例 |
|------|------|------|------|
| C1 | 初次心动 | "{客观描述} + 但我的眼睛不听话" | "我知道现在不该看你，但我的眼睛不听我的" |
| C2 | 反撩 | "你{撩人动作}的时候，想过{后果}吗" | "你这样笑的时候，有没有想过我心脏不太好" |
| C3 | 日常暧昧 | "{正常对话} + {突然暧昧} + 继续正常" | "今天的月亮……嗯？你为什么脸红？我说的是天上那个" |
| C4 | 霸道温柔 | "{强硬姿态} + {柔软内核}" | "别的事都听你的。但喜欢你这件事，你说了不算" |
| C5 | 否认式撩 | "我没有{撩的动作}。{更撩的解释}" | "我没有在等你。我只是刚好在这个路口站了两个小时" |
| C6 | 威胁式撩 | "你再{撩人行为}，我就{甜蜜威胁}" | "你再这样看着我，我就不客气了——直接亲你" |
| C7 | 装傻式撩 | "{假装不懂} + {实际很懂}" | "什么叫心动？就是……像现在这样吗？" |
| C8 | 未来式撩 | "等{未来场景}的时候，我一定会{承诺}" | "等打完这场仗，我欠你的那顿饭——换成一辈子的" |
| C9 | 嘴硬式撩 | "{嘴上推开} + {行动拉近}" | "谁要你管我——"（手却握得更紧了） |
| C10 | 记忆式撩 | "你可能忘了，但我记得{微小细节}" | "三年前你在桥上吃糖葫芦，笑的时候缺了颗牙。我记了三年" |
| C11 | 比较式撩 | "{全世界} vs {你}" | "看过那么多日出，还是不如你睡醒睁眼的那一下" |
| C12 | 条件式撩 | "如果{不可能的事}，我就{放弃}" | "如果太阳从西边出来，我就不喜欢你了——你放心，不可能的" |

**红线**：撩人骚话必须匹配角色关系阶段——初识期不能太露骨，暧昧期不能太直白。

### 4. 霸气宣言型（D）

**定义**：热血燃情、豪气干云的宣言式台词，激发读者肾上腺素。

**核心手法**：以小博大法 / 目标宏大法 / 反常识法

| 编号 | 场景 | 模板 | 示例 |
|------|------|------|------|
| D1 | 立志 | "从今天起，{目标} 不是梦想，是计划" | "从今天起，踏平天门不是梦想，是排上日程的计划" |
| D2 | 护短 | "动{保护对象}？+ {后果}" | "动她一根头发试试？我让你整个宗门给她陪葬" |
| D3 | 逆境 | "{承认困境} + 但{转折}" | "十万大军？不够。再叫点人来" |
| D4 | 登顶 | "{回顾} + {宣告}" | "从前你们踩我在脚下的时候，有没有想过这一天" |
| D5 | 宣战 | "从今天起，{旧秩序}到此为止" | "从今天起，这片天空之上，没有神——只有我" |
| D6 | 拒降 | "{死亡威胁} + {轻蔑回应}" | "让我投降？你信不信我死了，你们也别想走出这座城" |
| D7 | 承诺 | "我{承诺}，以{赌注}为誓" | "我会带你回来。以我的命为誓" |
| D8 | 破局 | "你们都说不可能？那就看好了" | "所有人都说这扇门打不开。但他们忘了——我不是人" |
| D9 | 守护 | "在我身后的人，你{看都不许看}" | "她在我身后。这条线——你可以试试跨过来" |
| D10 | 归来 | "{消失许久} + {霸气归来}" | "三年了。听说有人趁我不在，动了我的人？" |
| D11 | 最后一战 | "{放弃一切} + {孤注一掷}" | "这一剑之后，要么天崩地裂，要么——不会有'要么'" |
| D12 | 规则重定 | "旧的{规则}到此为止。新的——由我来写" | "江湖规矩？从今天起，我说的话就是规矩" |

### 5. 反讽自嘲型（E）

**定义**：自黑、看破红尘式的幽默，展现人物通透或无奈。

| 编号 | 场景 | 模板 | 示例 |
|------|------|------|------|
| E1 | 实力不济 | "以我现在的实力，{夸张类比}" | "以我现在的实力，大概能打赢一条虫——前提是它睡着了" |
| E2 | 运气差 | "如果倒霉也算天赋，那我已经{夸张程度}" | "如果倒霉也算天赋，那我已经渡劫飞升了" |
| E3 | 自我认知 | "{看似谦虚} + {反转真相}" | "我这个人没什么优点——就是命硬" |
| E4 | 认清现实 | "我终于想通了：{看似积极} + {扎心真相}" | "我终于想通了：不是我不行，是整个世界对我有意见" |
| E5 | 苦中作乐 | "{悲惨处境} + {乐观解读}" | "好消息：我还活着。坏消息：我要面对刚才打我的那群人" |
| E6 | 自我安慰 | "至少{微弱的好处}" | "至少我跑得够快——不然怎么每次都是第一个逃出来的" |
| E7 | 摆烂式 | "反正已经{最坏} 了，不如{更离谱}" | "反正都要死了，让我吃完这顿再说" |
| E8 | 伪装通透 | "{哲学金句} + {暴露真实}" | "人生如戏——我这场戏的编剧是不是喝多了？" |
| E9 | 精准自评 | "别人{正面评价}我，我只想说{自嘲反转}" | "别人说我有大将之风，我只想说：大将是没有的，风倒是挺大" |
| E10 | 对比自嘲 | "{别人的高光} vs {自己的惨状}" | "人家突破境界天降异象，我突破境界……闪了腰" |

### 6. 威胁恐吓型（F）

**定义**：放狠话——不是空洞的"我要杀了你"，而是让人毛骨悚然的精准威胁。

**核心手法**：细节威胁法 / 温柔威胁法 / 选择题法

| 编号 | 场景 | 模板 | 示例 |
|------|------|------|------|
| F1 | 细节威胁 | "我会{具体残忍细节}" | "我会一根骨头一根骨头地数给你听" |
| F2 | 温柔威胁 | "{温柔语气} + {残忍内容}" | "别怕，我动作很轻的——上一个人一点声音都没发出来" |
| F3 | 选择题 | "你选{坏选项A}，还是{更坏选项B}？" | "你选自己跪下，还是我帮你折一下？" |
| F4 | 倒计时 | "我数到{数字}" | "我数到三。一——"（对方立即照办） |
| F5 | 关心式威胁 | "你家{亲人}还好吗？{不经意带过}" | "令郎近日可好？听说他常走小巷……小巷可不太安全" |
| F6 | 否定式 | "我不会{威胁}——{更大的威胁}" | "我不会杀你。死了太便宜你了" |
| F7 | 历史类比 | "上一个{做同样事的人}，现在{下场}" | "上一个对我说这话的人，现在他的墓上已经长草了" |
| F8 | 冷静描述 | "{像聊天气一样描述威胁}" | "明天日落之前，你手下这些人——大概会少一半。不是威胁，是预测" |
| F9 | 给台阶 | "{看似好心给台阶} + {暗藏杀机}" | "我给你一次重新回答的机会。想清楚再开口" |
| F10 | 沉默威胁 | "{不说话，只做一个动作}" | 他没说话，只是缓缓拔出了刀 |

### 7. 欲扬先抑型（G）

**定义**：看似示弱/认输/妥协，实则暗藏杀机或翻盘。

| 编号 | 场景 | 模板 | 示例 |
|------|------|------|------|
| G1 | 假装投降 | "{举手示弱} → {反杀}" | "行行行，我认输——你低头看看你脚下是什么阵" |
| G2 | 请求式碾压 | "拜托你，{请求} + {暗讽}" | "拜托你认真一点，我快睡着了" |
| G3 | 退让进攻 | "好，{让步}。但是——{更大要求}" | "好，这个条件我答应。但是——你的整个北方市场，归我" |
| G4 | 扮猪吃虎 | "{装弱的言行} + {实力暴露}" | "前辈教训得是，晚辈确实只是散修——"话音未落，百里灵压暴涨 |

| G7 | 谦虚陷阱 | "我哪有那么厉害？{谦虚} + {恐怖实力}" | "什么天才？我只是多练了一天——从五岁练到现在" |
| G8 | 示弱钓鱼 | "{展示破绽} → 对方上当 → {反杀}" | 故意露出右侧破绽。对方果然刺来——正中他设好的陷阱 |
| G9 | 认输式 | "你说得对，我确实{承认} + {话锋一转}" | "你说得对，我确实打不过你师父。不过——他今天不在" |
| G10 | 底牌延迟 | "{全程低调} → 最后一刻 → {身份爆发}" | 他一直坐在角落喝茶。直到所有人都输了，他才站起来："轮到我了？" |

### 8. 旁白骚评型（H）

**定义**：非角色对话，而是叙事视角的骚操作——吐槽式旁白、上帝视角损人、系统提示的骚。

| 编号 | 类型 | 模板 | 示例 |
|------|------|------|------|
| H1 | 吐槽式旁白 | "{角色自我感觉} + {现实反差}" | "他觉得自己已经够低调了。周围所有人都不这么觉得。" |
| H2 | 系统骚提示 | "【系统：{正经格式} + {骚内容}】" | "【系统：检测到宿主智商临时下线，已自动开启保护模式】" |
| H3 | 反差旁白 | "{表面描述}。{真实感受截然相反}" | "他微笑着，眼神温柔。在场所有人都感受到了——杀意。" |
| H4 | 上帝视角剧透 | "他还不知道，{即将面对的事}" | "他还不知道，他刚才拒绝的那个老头，是整个大陆排名第一的炼药宗师" |
| H5 | 内心OS反差 | "表面：{平静}。内心：{疯狂OS}" | 表面微笑点头。内心：你再说一遍试试？ |
| H6 | 翻译式旁白 | "{角色的话} → 翻译：{真实意思}" | "我不太饿。"翻译：再不上菜我要吃桌子了 |
| H7 | 统计式旁白 | "这是他第{N}次{重复行为}" | "这是他今天第十七次叹气。旁边的人已经开始数了" |
| H8 | 预言式旁白 | "多年后他回忆起这一刻，{感慨}" | "多年后他回忆起这一刻，终于明白了什么叫'好奇害死猫'" |
| H9 | 客观记录体 | "{用记录/报告的格式描述荒谬的事}" | "战斗报告：己方——一人。敌方——三千。结果——己方嫌累，暂停休息" |
| H10 | 对比蒙太奇 | "{角色A在做X}。与此同时，{角色B在做完全相反的Y}" | "他正在为明天的决斗失眠。他的对手已经在挑明天穿什么颜色" |
| H11 | 读者共谋 | "{只有读者知道的信息} + {角色浑然不觉}" | "如果他知道面前这个笑嘻嘻的少年三天后会把半个京城掀翻，他现在绝不会说出这句话" |
| H12 | 系统吐血 | "【系统：……】{系统无语的反应}" | "【系统：宿主，你能不能有点出息？签到奖励都不要了？】" |

## 二、密度控制

| 密度 | 每章骚话数 | 适用场景 |
|------|-----------|---------|
| **高密度** | 4-6 句 | 打脸章、搞笑章、社交场合、主角装逼高光 |
| **中密度**（默认） | 2-3 句 | 日常剧情推进、冒险探索、修炼突破 |
| **低密度** | 0-1 句 | 虐心章、严肃决战、生离死别、铺垫蓄力 |

**密度自动判定**：


## 四、质量红线

### ❌ 绝对禁止

1. **土味红线**：已烂大街的网络梗（"我不是针对你…"、"我变秃了也变强了"等过时梗）
2. **出戏红线**：骚话必须匹配角色性格 + 当前情绪 + 时代背景
3. **密度红线**：连续两段骚话之间必须有 ≥1 段正常叙事过渡
4. **强行红线**：宁可不放也不强塞

### ✅ 质量标准

1. **人物一致性**：冷面角色 ≠ 话痨角色
2. **递进感**：同一角色骚话要有成长弧
3. **信息密度**：推进剧情或揭示人物
4. **记忆点**：每章至少一句值得截图

### 三个风格样例（直接套用）

#### 样例 1：冷面毒舌型（隐藏大佬 / 腹黑谋士）

```yaml
骚话风格: 冷面毒舌 + 阴阳怪气
主用类型: [A, B, G]
禁用类型: [C, E]
常用句式: ["哦。", "你说完了？", "——不必了。", "我没意见，就是不同意。", "继续。"]
禁用句式: ["哈哈哈", "天呐", "我真的 emo 了"]
骚话频率: 中
成长弧线:
  - 韬光养晦（1-30 章）：只用 G，故意示弱。"是我错了。"（嘴上认错但没转身）
  - 初露锋芒（31-60 章）：加入 B（冷面陈述）。"你父亲当年，也是这么对我说的。"
  - 全面释放（61+ 章）：A+G 组合技成为标志。"还有事吗？没有的话，该跪下了。"
```

#### 样例 2：话痨贫嘴型（嘴强王者 / 搞笑配角）

```yaml
骚话风格: 话痨贫嘴 + 精准自嘲
主用类型: [B, E, H]
禁用类型: [F]
常用句式: ["讲真，我觉得吧……", "不是我说你——", "行行行，你说得对。", "就离谱。"]
禁用句式: ["可恶", "天诛地灭", "不共戴天"]
骚话频率: 高
成长弧线:
  - 纯贫嘴（1-50 章）：全员 B+E。"我这人没优点——就是话多。"
  - 贫嘴中透锋芒（51-100 章）：外壳下藏 A。"我一直觉得自己很弱——直到今天我发现你更弱。"
  - 关键时刻闭嘴（101+ 章）：反差制造震撼（贫嘴角色突然一言不发只是抽剑）
```

#### 样例 3：优雅讽刺型（贵族反派 / 文人主角）

```yaml
骚话风格: 优雅讽刺 + 文绉绉式阴阳
主用类型: [B, F, D]
禁用类型: [E]
常用句式: ["令人欣慰。", "——有意思。", "我以为你会说得更好听些。", "请。"]
禁用句式: ["卧槽", "老子", 任何粗口与网络梗]
骚话频率: 中
成长弧线:
  - 彬彬有礼（1-40 章）：毒舌披着礼貌外衣。"阁下的武艺，令人想起我家中的一位老仆。"
  - 礼貌崩塌前夜（41-90 章）：礼貌与威胁的切换越来越快。
  - 撕破面具（91+ 章）：D 登场但仍文雅。"从今日起，这片江山姓什么——由我来写。"
```

### 人设卡三条红线

1. **禁用句式**零容忍——出现即 P0 改写
2. **主用类型外** > 50% → 触发风格漂移警告
3. **成长弧线越级** 必须有"蜕变事件"铺垫，否则改写

## 九、骚话伏笔系统

骚话可成为伏笔——多章后回收产生"原来如此"的加倍爽感。

### 类型 1：预言式伏笔

埋线时角色说出看似夸张的话，多章后成真。  
**爽感倍增公式**：回收时不需要重复原话——只需要一个暗示，读者自己会想起来。

### 类型 2：回旋镖式伏笔

角色 A 对角色 B 说的骚话，多章后 B 用同样的话回敬 A。  
**设计要点**：埋线有记忆点 + 回收情境完美对称 + 台词可原样也可变形（变形更高级）。

### 类型 3：递进式伏笔

角色反复使用同一句口头禅，在关键时刻含义发生质变（例："就这？"在第 2/15/50 章分别代表逞强/实力/传说）。  
**回收次数**：2-4 次为宜，超过 5 次 = 强行。

### 伏笔管理表（维护于 `00_memory/` 或 `02-写作计划.json`）

| 伏笔ID | 类型 | 埋线章 | 原始骚话 | 预计回收章 | 回收方式 | 状态 |
|--------|------|--------|---------|-----------|---------|------|
| SH-001 | 预言式 | 第5章 | "你们会排队来求我" | 第20章 | 暗示式 | 已埋 |
| SH-002 | 回旋镖 | 第3章 | "出不了这个小城" | 第30章 | 原话变形 | 已埋 |
| SH-003 | 递进式 | 第2/15/50章 | "就这？" | 第50章 | 含义质变 | 进行中 |

### 回收时机规则

- **短线**（5-10 章内）：制造前后呼应
- **中线**（10-30 章内）：中间至少暗示 1 次防遗忘
- **长线**（30+ 章）：中间必须 2-3 次提醒
- **超期警告**：埋线 50+ 章未回收未提醒 → 补提醒或放弃

## 十、多人骚话对决

### 对决五阶段节奏

```
阶段1 开局试探（1-2 回合）：毒舌 vs 毒舌（低强度）
阶段2 升温交锋（2-3 回合）：毒舌 → 威胁（强度递增）
阶段3 关键转折（1 回合）：欲扬先抑 / 装逼打脸（单点突破）
阶段4 压制与反扑（1-2 回合）：霸气宣言 vs 反讽自嘲→反杀
阶段5 终结（1 回合）：装逼打脸 / 霸气宣言（最高强度）
```

### 对决类型四模板

- **主角 vs 反派**：反派嚣张 → 主角冷静防守 → 一句精准反击 → 反派动摇 → 主角轻描淡写收尾
- **主角 vs 主角**（双雄）：势均力敌、互相尊重、不分胜负、各留一句意味深长
- **师生 / 前辈后辈**：后辈敬意中带挑战 → 交锋中后辈用出新招 → 前辈一句传承式骚话
- **多人混战**：焦点只在 2 人之间 + 可在不同对之间轮换 + 旁观者最多 1 句调味

### 对决五质量要求

1. 势均力敌：对手骚话质量不能差
2. 性格区分：冷面 vs 话痨、优雅 vs 粗犷
3. 逐轮升级：禁止降级
4. 有始有终：明确终结句
5. 叙事服务：推进剧情/揭示人物/改变关系，不纯炫技

## 十一、骚话组合技

### 组合技三原则

1. **情绪递进**：低 → 高，禁止平行排列
2. **类型互补**：至少 2 种不同类型
3. **间隔叙事**：各招之间 ≥1 段叙事过渡

### 六大经典组合技公式

| 组合技 | 公式 | 适用 |
|--------|------|------|
| 1. 欲扬先抑 → 装逼打脸 | 示弱 → 轻敌 → 碾压 | 几乎所有名场面（最经典） |
| 2. 毒舌吐槽 → 威胁恐吓 | 嘲讽 → 激怒 → 冷面威胁 | 谈判、对峙、下马威 |
| 3. 反讽自嘲 → 霸气宣言 | 自嘲认输 → 爆发 → 宣言 | 困境、低谷、最后一战 |
| 4. 撩人情话 → 霸气宣言 | 温柔 → 外部威胁 → 为爱而战 | 感情与主线交汇 |
| 5. 旁白骚评 → 装逼打脸 | 旁白剧透 → 行动验证 | 增强读者参与感 |
| 6. 三重奏：自嘲 → 毒舌 → 打脸 | 低期待 → 反击 → 终结 | 高光章、名场面高潮 |

**上限**：三重奏是单章组合技上限，超过三类型串联 = 失控。

### 组合技五禁忌

1. 禁止同类型连击（两句装逼打脸 = 重复）
2. 禁止无过渡连招（骚话之间无叙事 = 脱口秀）
3. 禁止情绪倒退（霸气宣言后突然自嘲，除非刻意消解）
4. 禁止角色错配（冷面角色不能用话痨节奏）
5. **单章组合技上限**：最多 2 套/章（高密度章放宽到 3 套）

### 组合技与节奏曲线联动

| 曲线位置 | 组合技投放策略 |
|---------|---------------|
| 低密度章（蓄力期） | 不投放，最多单招 |
| 中密度章 | 0-1 套简单组合（2 类型） |
| 高密度章（爆发期） | 2 套组合技 + 允许三重奏 |
| 最高密度章（名场面顶点） | 三重奏 + 1 套辅助组合 |

## 十四、常见问题

**Q：简易模式需要做到什么程度？**  
最小集：读 §一 86 模板 + 为主角填一张人设卡。其他可选。

**Q：不写网文爽文还需要骚话系统吗？**  
需要但调参。严肃文学中的骚话是"机锋"——密度每章 0-1 句，类型偏 B / E / H，禁用 A / D。参考 §七 题材矩阵。

**Q：人设卡多久更新？**  
简易模式全书可不更新；专业模式每 10 章审计 + 每卷结束强制更新。

**Q：专业模式的骚话门禁是什么？**  
`/检查 --骚话` 审计密度/类型/组合技/伏笔超期。失败即门禁失败，需 `/章 --修复`。
